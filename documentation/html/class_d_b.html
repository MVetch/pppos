Подробное описание

Перегружает некоторые функции стандартного класса mysqli.

Методы

◆ q()

q (   $select,
  $from,
  $where = array() 
)

Функция делает запрос в базу данных с заданными параметрами. Запрещается использовать для выборки, в которой участвует более чем 1 таблица. Для этого необходимо использовать функцию query()

Аргументы
array$selectПоля для выбора
string$fromТаблица, из которой идет выбор
array$whereАссоциативный массив, где ключами являются поля, а значениями необходимые для отсеивания значения в соответствующих полях (могут быть массивы для сравнения на равенство).
